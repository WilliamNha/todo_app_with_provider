# Flutter Todo app

A simple todo app with provider as state management.

## How to Use
- To use the app with api clone and run the api project with this link:
  https://github.com/WilliamNha/todo-task-api-nodejs

## All features 
- Create task
- Get all tasks
- Get task by ID
- Update task
- Delete task
- Toogle task (done || !done)
